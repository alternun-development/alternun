# นโยบายความเป็นส่วนตัว

**อัปเดตล่าสุด:** มกราคม 2025

## บทนำ

Alternun ("เรา") มุ่งมั่นที่จะปกป้องความเป็นส่วนตัวของคุณ นโยบายความเป็นส่วนตัวนี้อธิบายวิธีที่เรารวบรวม ใช้ และปกป้องข้อมูลของคุณเมื่อคุณเยี่ยมชมเว็บไซต์และใช้บริการของเรา

## ข้อมูลที่เรารวบรวม

### ข้อมูลส่วนตัว

- ชื่อและข้อมูลติดต่อ (อีเมล โทรศัพท์)
- ข้อมูลรับรองและข้อมูลการตรวจสอบ
- ข้อมูลการเงินสำหรับธุรกรรม
- รูปแบบการใช้งานและความชอบ

### ข้อมูลที่รวบรวมโดยอัตโนมัติ

- ที่อยู่ IP และข้อมูลอุปกรณ์
- ประเภทเบราว์เซอร์และระบบปฏิบัติการ
- หน้าที่เยี่ยมชมและเวลาที่ใช้ในไซต์ของเรา
- คุกกกี้และเทคโนโลยีการติดตามที่คล้ายกัน

## วิธีที่เราใช้ข้อมูลของคุณ

เราใช้ข้อมูลของคุณเพื่อ:

- ให้และรักษาบริการของเรา
- ประมวลผลธุรกรรมและส่งข้อมูลที่เกี่ยวข้อง
- สื่อสารกับคุณเกี่ยวกับบัญชีของคุณ
- ปรับปรุงผลิตภัณฑ์และประสบการณ์ของผู้ใช้
- ปฏิบัติตามภาระงับทางกฎหมาย

## การแชร์และเปิดเผยข้อมูล

เราไม่ขาย แลกเปลี่ยน หรือโอนข้อมูลส่วนตัวของคุณให้บุคคลที่สามโดยไม่ได้รับความยินยอมจากคุณ ยกเว้น:

- กับผู้ให้บริการที่เชื่อถือได้ซึ่งช่วยในการดำเนินงานเว็บไซต์ของเรา
- เมื่อกฎหมายกำหนดหรือเพื่อปกป้องสิทธิของเรา
- ในกรณีที่เกี่ยวข้องกับการโอนธุรกิจหรือการควบรวม

## ความปลอดภัยของข้อมูล

เราใช้มาตรการทางเทคนิคและองค์กรเพื่อปกป้องข้อมูลส่วนตัวของคุณจากการเข้าถึง การเปลี่ยนแปลง การเปิดเผย หรือการทำลายโดยไม่ได้รับอนุญาต

## คุกกกี้

เว็บไซต์ของเราใช้คุกกกี้เพื่อ:

- จดจำความชอบของคุณ
- เข้าใจวิธีที่คุณใช้ไซต์ของเรา
- ให้เนื้อหาที่ปรับแต่ง
- วิเคราะห์การจราจรของเว็บไซต์

คุณสามารถควบคุมคุกกกี้ผ่านการตั้งค่าเบราว์เซอร์ของคุณ

## สิทธิของคุณ

คุณมีสิทธิที่จะ:

- เข้าถึงข้อมูลส่วนตัวของคุณ
- แก้ไขข้อมูลที่ไม่ถูกต้อง
- ร้องขอให้ลบข้อมูลของคุณ
- คัดค้านการประมวลผลข้อมูลของคุณ
- การพกพาข้อมูล

## การถ่ายโอนข้อมูลระหว่างประเทศ

ข้อมูลของคุณอาจถูกถ่ายโอนและประมวลผลในประเทศที่แตกต่างจากของคุณ เรามั่นใจว่ามีมาตรการป้องกันที่เหมาะสม

## ความเป็นส่วนตัวของเด็ก

บริการของเราไม่ได้มุ่งสำหรับเด็กที่มีอายุต่ำกว่า 18 ปี เราไม่ได้รวบรวมข้อมูลส่วนตัวของเด็กที่มีอายุต่ำกว่า 18 ปีโดยเจตนาจริง

## การเปลี่ยนแปลงนโยบายนี้

เราอาจอัปเดตนโยบายความเป็นส่วนตัวนี้เป็นครั้งไป เราจะแจ้งให้คุณทราบการเปลี่ยนแปลงโดยการโพสต์นโยบายใหม่ในหน้านี้

## ติดต่อเรา

หากคุณมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ โปรดติดต่อเรา:

- อีเมล: privacy@alternun.io
- เว็บไซต์: https://alternun.io
