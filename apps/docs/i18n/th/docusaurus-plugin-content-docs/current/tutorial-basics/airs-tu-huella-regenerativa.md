# AIRS — รอยเท้าการฟื้นฟูของคุณ

ชื่อเสียงและการมีส่วนร่วมของคุณในระบบนิเวศ

ที่ Alternun เราไม่เชื่อว่าผลกระทบวัดได้จากเงินเท่านั้น

เราเชื่อว่า **มูลค่าที่แท้จริงอยู่ในสิ่งที่คุณทำ วิธีที่คุณมีส่วนร่วม และวิธีที่คุณรักษาระบบนิเวศไว้ได้ตลอดเวลา**

นั่นคือเหตุผลที่เราสร้าง **AIRS**

## AIRS คืออะไร?

AIRS คือ **ระบบชื่อเสียงและการมีส่วนร่วมของ Alternun**

มันคือ **คะแนนที่สะท้อนถึงการมีส่วนร่วมอย่างแท้จริงของคุณ** ต่อระบบนิเวศการฟื้นฟู

การกระทำทุกอย่างที่คุณทำ ภายใน Alternun — ค่าชดเชย สนับสนุนโครงการ รักษาทรัพย์สินการฟื้นฟู การเข้าร่วมความคิดริเริ่ม — เพิ่ม AIRS ให้กับบัญชีของคุณ

**ต่างจากระบบอื่น ๆ ไม่สามารถซื้อ AIRS ได้**

มันถูกสร้างขึ้นได้ตลอดเวลา ด้วยการกระทำที่ตรวจสอบได้และความมุ่งมั่น

## AIRS ใช้ทำอะไร?

AIRS เป็นพื้นฐานที่เชื่อมต่อระบบนิเวศทั้งหมด:

- **กำหนดสถานะของคุณ** ภายใน Alternun
- **กำหนดการมีส่วนร่วมของคุณ** ใน [Regenerative Basic Income (RBI)](/docs/tutorial-basics/rbi-regenerative-basic-income)
- **ให้คุณเข้าถึง** สิทธิ ส่วนลด และโอกาสภายในแพลตฟอร์ม
- **สร้างชื่อเสียงการฟื้นฟูของคุณ** มองเห็นได้และตรวจสอบได้

**กล่าวโดยสรุป: AIRS คือวิธีที่ Alternun ยอมรับผู้ที่ทำให้ระบบทำงาน**

## คุณจะได้ AIRS อย่างไร?

AIRS ได้รับการเรียนรู้จาก **การมีส่วนร่วมอย่างกระตือรือร้นในระบบนิเวศ**

บางวิธีที่พบบ่อยที่สุด ได้แก่:

- **ทำค่าชดเชยที่ตรวจสอบได้**
- **สนับสนุนโครงการฟื้นฟู** โดยการซื้อหรือถือ โครงการที่มีสิทธิ
- **เข้าร่วมความคิดริเริ่ม แคมเปญ หรือการกระทำ** ที่ส่งเสริมโดย Alternun และพันธมิตร
- **รักษากิจกรรมที่ยั่งยืนและรับผิดชอบ** ภายในแพลตฟอร์ม

แต่ละการกระทำมี **ค่า AIRS ที่กำหนดและโปร่งใส**

**ไม่มีทางลัดหรือการสะสมที่ไม่สมควร**

## AIRS เติบโตพร้อมกับคุณ

คะแนน AIRS ของคุณ **เติบโตตลอดไป**

**มันไม่ได้ใช้ไป ไม่ถูกเผา ไม่ถูกโอน**

เมื่อเวลาผ่านไป AIRS ช่วยให้คุณบรรลุสถานะต่างๆ ภายในระบบนิเวศ — จากระดับเริ่มต้นจนถึงระดับสูงสุด — ซึ่งสะท้อนถึงประวัติและความสม่ำเสมอของคุณ

**สถานะไม่ใช่เพียงสัญลักษณ์: มันช่วยให้เกิดความเป็นจริงและการมีส่วนร่วมมากขึ้นในมูลค่าที่ Alternun สร้างขึ้น**

## AIRS และเวลา

AIRS ถูกออกแบบ **เพื่อให้รางวัลแก่ความสม่ำเสมอ ไม่ใช่การเก็งกำไร**

มันไม่ใช่แค่สิ่งที่คุณทำครั้งเดียว แต่ **วิธีที่คุณยังคงมีส่วนร่วม**

นั่นคือเหตุผลที่ระบบนี้คำนึงถึง **กิจกรรมที่ยั่งยืนและความสม่ำเสมอตลอดเวลา**

วิธีนี้ทำให้ AIRS เป็นธรรมชาติสำหรับผู้ที่เริ่มต้นวันนี้และผู้ที่สนับสนุนระบบนิเวศมาหลายปี

## คนและองค์กร

AIRS สามารถเก็บสะสมได้โดย **บุคคล และ องค์กร**

ในทั้งสองกรณี AIRS ทำหน้าที่เป็น **เมตริกชื่อเสียงการฟื้นฟู:**

สัญญาณที่ชัดเจนของความมุ่งมั่น สามารถติดตามได้ และเปรียบเทียบได้ภายในระบบนิเวศ

นี่เปิดประตูสู่วิธีการทำงานร่วมกันใหม่ระหว่างบุคคล บริษัท และโครงการที่มีค่านิยมสอดคล้องกัน

## AIRS ไม่ใช่เพียงตัวเลข

**AIRS คือการบรรยายเกี่ยวกับผลกระทบ**

มันคือเรื่องราวของวิธีการมีส่วนร่วมของคุณในการฟื้นฟู และวิธีที่การมีส่วนร่วมนั้นแปลเป็นมูลค่าที่ใช้ร่วมกัน

**ที่ Alternun ผลกระทบของคุณมีความสำคัญ และ AIRS คือวิธีที่มันกลายเป็นที่มองเห็นได้**
