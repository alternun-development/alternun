# Regenerative Basic Income (RBI)

รายได้ที่เติบโตพร้อมกับระบบนิเวศของคุณ

ที่ Alternun เราเชื่อว่าการดูแลและฟื้นฟูโลกไม่ควรเป็นการกระทำที่โดดเดี่ยวหรือการสูญเสียทางเศรษฐกิจ นั่นจึงเป็นเหตุผลที่เราสร้างขึ้น **Regenerative Basic Income (RBI)**: รายได้พาสซีฟที่เกิดขึ้นจากกิจกรรมการฟื้นฟูชีวิตที่เกิดขึ้นภายในระบบนิเวศ

RBI ไม่ใช่สัญญาในอนาคตหรือเงินอุดหนุนภายนอก **มันคือผลจากการมีส่วนร่วมอย่างสักสรรพในการดำเนินงาน Alternun**

## RBI คืออะไร?

Regenerative Basic Income คือรายได้แบบปกติที่แจกจ่ายให้กับผู้ใช้ Alternun ที่มีส่วนร่วมในระบบนิเวศ

มันไม่ได้ขึ้นอยู่กับว่าคุณมีเงินเท่าใด แต่ขึ้นอยู่กับ **การมีส่วนร่วม ความมุ่งมั่น และผลกระทบของคุณ** วัดโดยคะแนน [AIRS](/docs/tutorial-basics/airs-tu-huella-regenerativa) ของคุณ

ทุกครั้งที่ Alternun ช่วยจัดหาค่าชดเชย การขายจากกลุ่มค่าชดเชย หรือกิจกรรมทางเศรษฐกิจใด ๆ ภายในแพลตฟอร์ม ส่วนหนึ่งของมูลค่านั้นจะไปยังกองทุน RBI โดยตรง

**กล่าวอีกนัยหนึ่ง เมื่อระบบนิเวศเติบโตและมีการใช้ RBI ก็จะเติบโตไปด้วย**

## RBI ได้รับการสนับสนุนอย่างไร?

กองทุน RBI ได้รับการจัดหาจากแหล่งต่างๆ หลายแหล่งภายในระบบ:

- **ส่วนหนึ่งของเบี้ยประกันภัย** ที่สร้างขึ้นจากค่าชดเชยที่ทำผ่าน Alternun
- **รายได้จาก** การขายโครงการส่วนที่เป็นเศษส่วนของโครงการชดเชยจากกลุ่มค่าชดเชย
- **ค่าธรรมเนียมโครงสร้างพื้นฐาน** และการดำเนินการเฉพาะภายในแพลตฟอร์ม
- **การมีส่วนร่วมที่ผิดปกติ** จากมูลนิธิ ตัวแทนทางยุทธศาสตร์ หรือสิ่งจูงใจพิเศษของระบบนิเวศ

การออกแบบนี้ทำให้ RBI **มีความแปรปรวนและปลั่นปลอด** มีเดือนที่มีรายได้สูงกว่า — จากแคมเปญ การเป็นหุ้นส่วน หรือเงินอุดหนุน — และเดือนที่เงียบสงบกว่า ทุกสิ่งคือ **มองเห็นได้และโปร่งใส** สำหรับชุมชน

## RBI จะจ่ายเมื่อไรและในอะไร?

RBI ถูกจ่ายทุกเดือน **ใน [ATN](/docs/tutorial-basics/atn-la-unidad-de-valor-del-ecosistema)** โครงการโทเคน Alternun

ที่จุดสิ้นสุดของแต่ละรอบรายเดือน ระบบจะคำนวณ:

1. เงินทั้งหมดที่มีอยู่ในกลุ่ม RBI
2. จำนวน AIRS ที่ใช้งานอยู่ทั้งหมดที่เป็นของผู้ใช้ที่มีสิทธิ
3. **อัตรา**: ATN จำนวนเท่าใดที่ตรงกับแต่ละจุด AIRS

จากนั้นผู้ใช้แต่ละคนสามารถเรียกร้องส่วนแบ่งตามสัดส่วนของตนได้ โดยอิงจาก AIRS สะสมของตนโดยเฉพาะเท่านั้น

**ไม่มีการจับฉลาก ไม่มีการตัดสินใจแบบกำหนด ไม่มีอคติ**

การคำนวณคือ **คณิตศาสตร์ สามารถตรวจสอบได้ และยุติธรรม**

## ใครสามารถรับ RBI?

ในการเข้าร่วมใน RBI คุณต้องปฏิบัติตามกฎง่ายๆ และชัดเจน:

- **มีบัญชี AIRS ที่ใช้งานอยู่**
- **รักษาการมีส่วนร่วมขั้นต่ำ** ในระบบนิเวศ ซึ่งอาจรวมถึง:
  - มี AIRS สะสม
  - รักษาการเก็บ ATN ขั้นต่ำในช่วงเวลานั้น
  - ปฏิบัติตามกฎสิทธิ์ของรอบปัจจุบัน

RBI **ไม่จ่ายให้กับบัญชีที่ไม่ใช้งานหรือเก็งกำไร** มันออกแบบมาเพื่อให้รางวัลแก่ผู้ที่สร้าง ขยาย และใช้ระบบนิเวศ

## เหตุใด RBI จึงแตกต่างออกไป?

ต่างจากรูปแบบ "รายได้พาสซีฟ" อื่น ๆ RBI ของ Alternun:

- **ไม่ขึ้นอยู่กับการขยายตัวของสินเชื่อและกำไรจากโครงการอย่างไม่สมควร**
- **ไม่ต้องการสัญญาผลตอบแทนที่กำหนดไว้**
- **ไม่อิงตามหนี้หรือความคาดหวังที่ไม่สมจริง**

RBI มีอยู่เพราะว่ามี **กิจกรรมจริง**: ค่าชดเชยจริง โครงการจริง ผลกระทบจริง

**รายได้ของคุณไม่ได้มาจากการสูญเสียของคนอื่น แต่มาจากการทำงานของระบบนิเวศได้อย่างดี**

## รายได้ที่สะท้อนถึงความมุ่งมั่น

RBI **ไม่ได้แทนที่การทำงาน** หรือสัญญาความมั่งคั่งทันที

มันลึกซึ้งกว่านั้น: **การยอมรับทางเศรษฐกิจของความมุ่งมั่นที่ยั่งยืนต่อการฟื้นฟู**

- การมีส่วนร่วมมากขึ้น = AIRS สูงขึ้น
- AIRS สูงขึ้น = การมีส่วนร่วมใน RBI มากขึ้น
- **ที่ Alternun การฟื้นฟูยังสร้างมูลค่าให้แก่คุณ**

---

## เริ่มสร้าง RBI ของคุณโดยการฟื้นฟูโลก

การมีส่วนร่วมของคุณใน Alternun คือขั้นตอนแรก การกระทำทุกอย่าง โครงการทุกโครงการที่คุณสนับสนุน ค่าชดเชยทุกอย่างที่คุณทำ สร้าง AIRS ของคุณและด้วยเหตุนี้ RBI ในอนาคตของคุณ
