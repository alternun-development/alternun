# ATN — หน่วยค่าของระบบนิเวศ Alternun

วิธีการหมุนเวียนและการกระจายมูลค่าในระบบนิเวศของเรา

ที่ Alternun ผลกระทบไม่เพียงแต่ได้รับการยอมรับ: มันยังได้รับการจัดหา แลกเปลี่ยน และกระจายใหม่

สำหรับสิ่งนั้นมี **ATN**

**ATN คือหน่วยค่าที่อนุญาตให้ระบบนิเวศทั้งหมดทำงานในวิธีที่ประสานงานและโปร่งใส**

มันไม่ใช่แนวคิดทางเทคนิคหรือสิ่งที่สงวนไว้สำหรับผู้เชี่ยวชาญ

**มันเป็นเพียงวิธีที่ Alternun แสดงและกระจายมูลค่าภายในแพลตฟอร์ม**

## ATN คืออะไร?

ATN คือ **สินทรัพย์ดิจิทัลของระบบนิเวศ Alternun** ที่ใช้สำหรับ:

- **รับรางวัล**
- **เข้าร่วมในโครงการ**
- **เข้าถึงสิทธิประโยชน์**
- **แลกเปลี่ยนมูลค่า** สำหรับบริการ ข้อดี หรือ หากคุณต้องการ เงิน

ลองคิดว่า ATN เป็น **คะแนนในโปรแกรมรางวัล** แต่มีความแตกต่างที่สำคัญ:

**ATN มีมูลค่าจริงภายในระบบและสามารถใช้ได้หลายวิธี**

## ATN ใช้ทำอะไร?

ATN มีบทบาทสำคัญหลายประการ:

- **มันเป็นวิธีการที่** [Regenerative Basic Income (RBI)](/docs/tutorial-basics/rbi-regenerative-basic-income) ได้รับการจ่าย
- **ช่วยให้คุณสามารถเข้าร่วม** ในโครงการและกิจกรรมภายในระบบนิเวศ
- **ใช้เพื่อเข้าถึง** ส่วนลด เงื่อนไขการสนับสนุน และโอกาสเฉพาะสิทธิ
- **ทำหน้าที่เป็นยานพาหนะ** ที่ Alternun กระจายมูลค่าที่สร้างขึ้นโดยกิจกรรมการฟื้นฟู

**ATN ไม่มีอยู่เพื่อที่จะสะสมโดยไม่มีวัตถุประสงค์**

**มันมีอยู่เพื่อหมุนเวียน ประสานงาน และเสริมสร้างระบบนิเวศ**

## คุณจะได้ ATN อย่างไร?

ATN ได้มาในหลายวิธีภายใน Alternun:

- **เป็นผลจาก RBI** ตามค่า [AIRS](/docs/tutorial-basics/airs-tu-huella-regenerativa) ของคุณ
- **ผ่านรางวัล** สำหรับการเข้าร่วมในระบบนิเวศ
- **ผ่านการแลกเปลี่ยน** ภายในแพลตฟอร์ม
- **โดยการเข้าร่วม** ในสิ่งริเริ่มหรือโครงการเฉพาะ

ในทุกกรณี **ATN มาเป็นผลจากการกระทำหรือการมีส่วนร่วมที่ชัดเจน**

ATN ยังสามารถซื้อได้เพื่อให้คุณสามารถทำการกระทำต่างๆ ภายในแพลตฟอร์ม

## ATN และการดุลยภาคของระบบนิเวศ

ส่วนสำคัญของการออกแบบ ATN คือ **ปริมาณของมันถูกจัดการอย่างมีความรับผิดชอบ**

ทุกครั้งที่ระบบนิเวศสร้างมูลค่า ส่วนหนึ่งของมูลค่านั้นจะใช้ **เพื่อลดปริมาณ ATN ในการหมุนเวียนอย่างก้าวหน้า** เสริมสร้างสมดุลของระบบและให้ประโยชน์แก่ผู้เข้าร่วมทั้งหมด

**สิ่งนี้ช่วยให้มั่นใจว่า ATN ไม่ใช่เพียงจุดสะสมเท่านั้น แต่เป็นหน่วยค่าที่สะท้อนถึงความเป็นอยู่ของระบบนิเวศ**

## ฉันสามารถทำอะไรกับ ATN ของฉัน?

ด้วย ATN คุณสามารถ:

- **เข้าถึงสิทธิประโยชน์** ภายใน Alternun
- **เข้าร่วมในโครงการฟื้นฟู**
- **รับเงื่อนไขที่ดีขึ้น** บนแพลตฟอร์ม
- **แปลงมูลค่าเป็นรางวัลหรือรายได้** เป็นดอลลาร์ ตามที่คุณต้องการ

**ATN มีความยืดหยุ่นตามการออกแบบ:**

สามารถใช้ แลกเปลี่ยน หรือถือไว้ได้ เสมอภายในกฎที่ชัดเจนและโปร่งใส

## ATN เป็นสะพาน

ATN **เชื่อมต่อสองโลก:**

- **โลกของการกระทำในแต่ละวัน** ความมุ่งมั่น และการมีส่วนร่วม
- **โลกของการกระจายมูลค่าใหม่** แรงจูงใจ และสิทธิประโยชน์

**ผ่าน ATN Alternun สามารถทำงานเป็นระบบที่ผลกระทบสร้างมูลค่า และมูลค่ากลับไปยังผู้คน**

## เริ่มสะสม ATN วันนี้
